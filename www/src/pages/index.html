Welcome to Brut

Brut is a brutalist static site generator.

It shares some of the principles behind brutalist architecture:

  • Raw: embrace raw materialsâ€”concrete for buildings, html and js for the web
  • Functional: don't do any more than necessary
  • Dependable: focus on stability, move slow and don't break things

It is designed to help build static websites out of a collection of raw js (or ts), html, and markdown files (plus static assets). Nothing more, nothing less.
